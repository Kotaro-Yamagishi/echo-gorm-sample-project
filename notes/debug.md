| 操作            | ショートカット                                                        |
| ------------- | -------------------------------------------------------------- |
| デバッグ開始/続行     | `F5`                                                           |
| デバッグ停止        | `Shift + F5`                                                   |
| ステップオーバー      | `F10`                                                          |
| ステップイン        | `F11`                                                          |
| ステップアウト       | `Shift + F11`                                                  |
| ブレークポイントの切り替え | `F9`                                                           |
| デバッグコンソールを開く  | `Ctrl + Shift + Y`（Windows/Linux）<br> `Cmd + Shift + Y`（macOS） |
| 一時停止（中断）      | `Ctrl + Shift + F5`                                            |
